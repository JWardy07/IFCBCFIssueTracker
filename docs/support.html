App: IFC BCF Issue Tracker

IFC BCF Issue Tracker helps you import, review, edit, comment on, and export BIM Collaboration Format issue files.

Supported File Types

The app supports `.bcf` and `.bcfzip` files, including BCF 2.1 and BCF 3.0 workflows.

Common Help

Use the Import BCF button to select a file from the Files app. After import, open the issues list to search, filter, edit issue details, add comments, view snapshots, and export reports.
